## Patent Management System

## About the Project

The Patent Management System is a web-based application designed to manage and organize patent-related information. It allows faculty to store patent details, view existing records, and manage them efficiently in one place. The main goal of this project is to demonstrate how a full-stack application works by connecting a frontend interface with a backend server and a database.
This project is built using **React (Vite)** for the frontend and **Node.js with Express** for the backend, while **MongoDB** is used to store the patent data.

## Project Status

This project is currently **under development (Work in Progress)**. New features and improvements are being added as the development continues.
